I am a motivated developer with experience building web applications from front to back. Over the past few years, I have worked on 4 web projects leveraging my skills in both frontend and backend development. Most recently, I built SITCODES, an e-learning platform using HTML, CSS, JavaScript, jQuery and Tailwind CSS for the frontend. For the backend, I utilized PHP and the Laravel framework to create a robust and scalable platform.

Prior to that, I developed StepOutInStyle, an e-commerce website selling fashion products online. I crafted an engaging frontend experience using HTML, CSS and modern JavaScript. The backend was powered by Laravel and PHP, allowing me to incorporate advanced features like user authentication, shopping carts, and inventory management.

I also created ExploreNepal360, a content management system for a travel company using Laravel and Vue.js. This gave non-technical users an easy way to publish articles and manage site content.

Additionally, I built a simple note taking app called Notes to improve my JavaScript skills. This single page application stored data in the browser using IndexedDB and had smooth drag and drop functionality.

Through these projects, I strengthened both my frontend development abilities like UI design, client-side interactivity, animations and frameworks like TailwindCSS and Vue as well as backend skills in APIs, databases, security and frameworks like Laravel. I am eager to apply this well-rounded skillset to new challenges as part of a fast-paced development team.
